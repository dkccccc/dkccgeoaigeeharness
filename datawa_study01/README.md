# datawa_study01 · 위성영상 뷰어

1장의 결과물. **지도에서 영역을 그리면, 그 영역의 최근 위성영상을 보여주는 뷰어**입니다.

## 만든 방법 (뼈대 → 기능 → 점검)

[PROMPTS.md](../PROMPTS.md)의 프롬프트 **1-1 → 1-2 → 1-3** 순서로 Claude Code에게 시켜 단계별로 키웠습니다.
한꺼번에 만들지 않고 한 번에 하나씩 — 그래야 어디가 잘못됐는지 보입니다.

## 실행 (주피터/코랩 권장)

```bash
pip install geemap earthengine-api jupyter
jupyter notebook   # 또는 Colab에 viewer.py 내용을 붙여넣기
```

`viewer.py` 의 코드를 노트북 셀에 붙여넣고 실행하면 지도가 뜹니다.

## 사용법

1. 지도 왼쪽의 사각형/다각형 그리기 도구로 관심 영역을 그린다.
2. `show_for_roi(m.user_roi)` 를 실행한다.
3. 그 영역의 구름 적은 최근 영상이 지도에 겹쳐 보인다.

## 확인 (이게 보이면 성공)

- 내가 그린 영역 안에만 위성영상이 또렷하게 보인다.
- 구름 낀 영상이 아니라 비교적 맑은 영상이 선택된다 (프롬프트 1-3으로 검증).

## 막히면

- `user_roi` 가 `None` → 아직 영역을 안 그렸거나, 그린 뒤 셀을 다시 실행하지 않은 경우.
- 영상이 안 보임 → 기간(`filterDate`)에 맑은 영상이 없을 수 있음. 기간을 넓혀보세요.
