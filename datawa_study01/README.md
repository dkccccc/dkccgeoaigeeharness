# datawa_study01 · 위성영상 뷰어 웹앱

1장의 결과물. **사이드바에서 지역(위도·경도·반경)을 고르면, 그 지역의 맑은 위성영상을 보여주는** Streamlit 웹앱입니다. 0장의 '영상 한 장' 앱을 키운 것입니다.

## 만든 방법 (뼈대 → 기능 → 점검)

[PROMPTS.md](../PROMPTS.md)의 프롬프트 **1-1 → 1-2 → 1-3** 순서로 Claude Code에게 시켜 `app.py`를 단계별로 키웠습니다.
한꺼번에 만들지 않고 한 번에 하나씩 — 그래야 어디가 잘못됐는지 보입니다.

- **1-1 뼈대**: 사이드바 + 지도를 띄우는 웹앱 골격
- **1-2 기능**: 고른 영역의 위성영상 + 구름 필터
- **1-3 점검**: 구름 필터 전후 비교

## 실행

```bash
pip install -r requirements.txt
earthengine authenticate     # 처음 한 번만 (브라우저 로그인)
streamlit run app.py
```

실행하면 **브라우저가 자동으로 열립니다.**

## 사용법

1. 왼쪽 사이드바에서 중심 위도·경도·반경(km)을 정한다.
2. (선택) 기간을 바꾼다.
3. **[분석 / 보기]** 버튼을 누른다.
4. 그 영역의 구름 적은 가장 맑은 영상이 지도에 보인다.

## 확인 (이게 보이면 성공)

- 사이드바에서 고른 영역에 위성영상(자연색)이 또렷하게 보인다. 🎉
- 구름 낀 영상이 아니라 비교적 맑은 영상이 선택된다 (프롬프트 1-3으로 검증).

## 막히면

에러 메시지를 복사해 Claude Code에게 *"이게 무슨 뜻이고 어떻게 고쳐?"* 라고 물어보세요. 자주 나는 것:

- `streamlit: command not found` → `pip install -r requirements.txt` 후 다시
