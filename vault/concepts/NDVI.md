# NDVI — 식생을 숫자로 보기

> [!question] 왜 필요한가
> 위성영상만 봐선 "여기 식물이 얼마나 건강한가"를 한눈에 알기 어렵다. 변화탐지의 핵심 단서인 **식생**을 숫자 하나로 요약하고 싶다.

## 한마디로
**NDVI(정규식생지수)** = 식물이 얼마나 무성한지를 −1~+1 숫자로 나타낸 값. 높을수록(초록) 식생이 건강하고, 낮으면(물·도시·맨땅) 식생이 적다.

## 조금 더
- 식물은 **근적외선**을 강하게 반사하고 **빨강**을 흡수한다. 이 차이를 이용한다.
- 공식: `NDVI = (근적외선 − 빨강) / (근적외선 + 빨강)`. 외울 필요 없다 — Claude에게 "NDVI 계산해줘"라고 하면 된다.
- 대략: 물 < 0, 도시·맨땅 ≈ 0~0.2, 풀·농경지 ≈ 0.3~0.6, 울창한 숲 ≈ 0.7+

> [!tip] 우리 프로젝트에서는
> 2장에서 NDVI로 식생지도를 만들고, 3장에서 **두 시점의 NDVI 차이**로 [[change-detection|변화를 탐지]]한다.

> [!warning] 검증하기
> NDVI가 맞는지 보려면 물(낮음)·숲(높음)을 비교한다. 물이 높게 나오면 뭔가 틀린 것. → [[hallucination-check|할루시네이션 검증]]
