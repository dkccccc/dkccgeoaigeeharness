# 바이브 코딩이란

> [!question] 왜 필요한가
> 위성영상 분석을 하고 싶은데 파이썬도, GEE API도 모른다. 코딩을 배우는 데만 몇 달이 걸릴 것 같다.

## 한마디로
**AI와 대화하며 코딩하는 방식.** 마치 말귀 알아듣는 조수에게 "이런 걸 만들어줘"라고 부탁하듯, 일상 언어로 요청하면 AI가 코드를 써준다. 여러분은 코드를 외우는 대신 **무엇을 원하는지 정확히 말하는 법**을 익힌다.

## 조금 더
전통적 코딩은 사람이 문법을 외워 코드를 직접 쓴다. 바이브 코딩은 사람이 *의도*를 말하고 AI가 *문장(코드)*을 쓴다. 사람의 역할은 바뀐다:
- **요청한다** — 무엇을, 왜 ([[프롬프트-4단계]])
- **확인한다** — 정말 동작하는지, AI가 틀리진 않았는지 ([[할루시네이션-검증]])
- **고친다** — 다시 말해서 다듬는다

> [!tip] 우리 프로젝트에서는
> 0장부터 6장까지, 변화탐지 서비스의 모든 코드를 Claude Code에게 말해서 만든다. 여러분이 GEE 명령을 외울 일은 없다.
