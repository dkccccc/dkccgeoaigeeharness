# 말로 만드는 위성영상 변화탐지 서비스

### — 클로드 코드 바이브 코딩으로 (원격탐사 연구자용 실습 교재)

> **한 줄 약속:** 코딩을 몰라도, **일상 언어로 Claude Code에게 말해서** 내 연구 주제의 서비스를 직접 만든다.

이 교재는 [혼자 공부하는 바이브코딩 with 클로드코드](https://github.com/taehojo/vibecoding)의 교수법을 **원격탐사 도메인**에 이식했습니다. 매 장마다 **돌아가는 결과물**을 만들고, 장이 쌓이면 *지도에 영역을 그리면 위성영상으로 변화를 보여주는 변화탐지 서비스*가 완성됩니다.

대상은 **원격탐사 초급 연구자** — 프로그래밍 경험이 적어도 괜찮습니다. 여러분이 외울 것은 GEE API가 아니라, **무엇을 원하는지 정확히 말하는 법**입니다.

---

## 📖 이 교재의 사용법

이 교재의 본문은 **Obsidian 위키**입니다. 노트끼리 `[[위키링크]]`로 연결돼 있어, 실습 중 모르는 개념을 클릭 한 번으로 펼쳐볼 수 있습니다.

1. [Obsidian](https://obsidian.md) 설치 → `vault/` 폴더를 볼트로 열기
2. `vault/00-홈.md` 에서 시작 (전체 지도)
3. 장별 실행 코드는 `datawa_study00/` ~ `datawa_study06/` 에 있습니다
4. 모든 프롬프트는 [PROMPTS.md](./PROMPTS.md) 에 모여 있습니다 — 복붙해서 바로 쓰세요

설치가 막히면 → [설치가이드](./설치가이드/README.md)

## 🗺️ 목차 — 매 장이 하나의 서비스

| 장 | 제목 | 🟢 만드는 것 | 핵심 |
|----|------|------------|------|
| **0** | 첫 바이브 코딩 | 위성영상 한 장을 띄우는 작은 페이지 | "말로 시키면 만들어진다"는 첫 성공 |
| **1** | 프롬프트가 전부다 | 영역을 그리면 위성영상을 보여주는 **뷰어** | 좋은 프롬프트 · 뼈대→기능→디자인→점검 |
| **2** | 클로드 코드 실전 | 식생 상태를 색으로 보여주는 **NDVI 앱** | 점진 개발 · 에러 함께 고치기 · 할루시네이션 검증 |
| **3** ⭐ | 변화를 탐지하기 | 작년 vs 올해 변화 **히트맵 앱** | 변화탐지를 말로 구현 · 결과 해석 |
| **4** | 진짜 데이터 붙이기 | 구름필터·기간선택 되는 **변화탐지 서비스** | 외부 데이터/API 연결 |
| **5** | AI 에이전트 팀 | 여러 지역을 자동 분석하는 **에이전트 팀** | 혼자가 아니라 팀에게 시키기 (Harness) |
| **6** | 세상에 내보내기 | **배포 링크** / MCP 확장 | 남에게 보여줄 수 있는 결과물 |
| **마무리** | 더 나아가기 | 내 연구 주제로 확장 | 실제 서비스는 어떻게 다른가 |

⭐ 3장이 관통 프로젝트의 심장입니다.

## 🧰 무엇을 쓰나요

| 도구 | 역할 | 비유 |
|------|------|------|
| **Claude Code** | 일상 언어를 코드로 바꿔주는 AI 개발 파트너 | 말귀 알아듣는 조수 |
| **Google Earth Engine (GEE)** | 전 세계 위성영상 창고 + 분석 엔진 | 위성 도서관 |
| **GeoAI** | 위성영상에서 "변화"를 읽어내는 분석 | 영상 판독관 |
| **Harness** | 여러 AI 에이전트를 팀으로 부리는 방식 | 연구팀 꾸리기 |
| **Obsidian** | 이 교재(위키)를 읽고 내 노트를 쌓는 곳 | 연결된 노트장 |

GEE 실습은 **Python + [geemap](https://geemap.org)** 으로 진행합니다 (Jupyter/Colab).

## ⚠️ 시작 전 한 가지

이 교재는 **여러분이 코드를 직접 짜기를 기대하지 않습니다.** 대신 Claude Code에게 *무엇을, 왜* 원하는지 말하고, 나온 결과를 **확인하고 고치는** 법을 익힙니다. AI는 가끔 그럴듯한 거짓말(할루시네이션)을 합니다 — 그래서 매 장에 "확인" 단계가 있습니다.

> 이 저장소는 교재 본문(`vault/`)과 실습 코드(`datawa_study*`)뿐 아니라, 교재를 만들어내는 **하네스**(`.claude/`)도 함께 담고 있습니다. 교재 자체가 "AI 팀으로 콘텐츠를 만드는" 예시입니다.
