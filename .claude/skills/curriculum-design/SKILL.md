# Curriculum Design (커리큘럼 설계)

통합 프로젝트(원격탐사 변화탐지 앱)를 관통 실습으로 삼아 GEE → GeoAI → Claude Code → Harness를 배우는 교재의 학습 경로와 Obsidian 볼트 골격을 설계한다.

## 설계 원칙

- **하나의 관통 프로젝트**. 4개 주제를 독립 모듈로 쪼개지 않는다. 변화탐지 앱을 만드는 자연스러운 빌드 순서에 주제를 녹인다.
- **실습이 목적, 개념은 수단**. 모든 챕터는 손으로 만든 결과물로 끝난다. 개념은 그 실습을 이해할 만큼만.
- **단조 증가 난이도**. 한 챕터에서 새로 도입하는 도구/개념 수를 제한(권장 1~2개). 선행 챕터 산출물이 다음 챕터 입력이 되도록 의존을 한 줄로 잇는다.
- **초급자 가정**. 프로그래밍 경험이 적은 원격탐사 연구자. 계정 생성·설치·인증부터 빠뜨리지 않는다.

## 권장 학습 경로 (관통 프로젝트의 빌드 순서)

기본 골격. 사용자 요청에 맞게 가감한다.

1. **오리엔테이션** — 우리가 만들 것(변화탐지 앱)과 도구 4종의 역할 한눈에. 환경 준비(Claude Code, Obsidian, GEE 계정).
2. **GEE 기초** — Earth Engine으로 위성영상 확보·전처리. 관심지역(AOI)·시기 선택, 화면에 띄우기.
3. **GeoAI 분석** — 두 시점 영상으로 변화탐지/분류 수행. 결과 해석.
4. **Claude Code 자동화** — AI 코딩(바이브 코딩)으로 위 분석을 재현 가능한 파이프라인/스크립트로 만든다.
5. **Harness 오케스트레이션** — 에이전트 팀으로 분석을 자동화·확장한다(메타 학습: 학습자가 쓰는 도구 자체를 다룬다).
6. **마무리** — 통합 회고, 다음 단계, 자기 프로젝트로 확장.

> 참고 프로젝트 `C:\dev\dkchangedetection`, `C:\dev\dkidi`는 실제 변화탐지/GEE 코드의 출처다. 실습 소재와 코드 패턴의 근거로 삼는다.

## Obsidian 볼트 구조 규약

```
vault/
├── 00-MOC.md                  ← 메인 진입점 (전체 지도)
├── 00-시작하기.md              ← 환경 준비
├── concepts/                  ← 개념 노트 (content-author)
│   ├── gee/  geoai/  claude-code/  harness/
├── labs/                      ← 실습 노트 (lab-builder)
│   ├── 01-gee/ 02-geoai/ 03-claude-code/ 04-harness/
└── assets/                    ← 이미지·코드·데이터
```

- **명명 규칙**: 실습은 순번 접두사(`lab-01-aoi-선택.md`)로 순서를 드러낸다. 개념은 개념명(`정사보정.md`). 공백 대신 하이픈, 한글 허용.
- **MOC**: 챕터별 인덱스 노트 + 메인 MOC. 모든 노트는 최소 1개 MOC에서 도달 가능해야 한다.
- **위키링크 토폴로지**: 실습 노트는 자신이 의존하는 개념 노트로 링크. 개념 노트는 관련 개념·해당 개념을 쓰는 실습으로 링크. 고아 노트 금지.
- **태그**: `#주제/gee` `#주제/geoai` `#주제/claude-code` `#주제/harness`, `#유형/개념` `#유형/실습`, `#난이도/입문` 등.

## 산출물 (모두 `_workspace/01_curriculum/`)

### `00_outline.md`
챕터 목록, 각 챕터의 학습 목표(동사로 시작: "~할 수 있다"), 선수 지식, 주제 매핑, 챕터 산출물.

### `01_vault-structure.md`
위 볼트 구조의 구체화 — 실제 폴더 트리, MOC 목록과 각 MOC가 인덱싱할 노트, 명명/링크/태그 규칙 확정본.

### `02_note-manifest.md` (가장 중요 — 다른 에이전트의 작업 명세서)
생성할 모든 노트를 표로:

| 경로 | 유형 | 담당 | 한 줄 요약 | 의존(선행 노트/실습) | 거는 위키링크 |
|------|------|------|-----------|--------------------|--------------|
| labs/01-gee/lab-01-aoi-선택.md | 실습 | lab-builder | AOI와 기간 선택해 영상 띄우기 | (없음) | [[좌표계]] [[정사보정]] |
